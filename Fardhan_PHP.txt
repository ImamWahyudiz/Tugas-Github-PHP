Figma: Figma adalah alat desain berbasis cloud yang digunakan untuk membuat

Git: Git adalah sistem kontrol versi yang membantu pengembang melacak perubahan
pada kode, berkolaborasi dengan orang lain, dan mengelola berbagai versi proyek

GitHub: GitHub adalah platform berbasis web untuk menyimpan repositori Git. 
GitHub menyediakan alat untuk kolaborasi, review kode, pelacakan masalah,

perangkat lunak.

VS Code (Visual Studio Code): VS Code adalah editor kode gratis dan open-source

memiliki banyak ekstensi, serta fitur seperti debugging, penyorotan sintaks, 
dan terminal terintegrasi untuk meningkatkan produktivitas pengembang.
